## Instalación
Clona este repositorio: 
```
git clone https://github.com/BarmanDev/HTML-Forms
```
Instala las dependencias:
```
npm install
```
Ejecuta la aplicación:
```
node app.js
```
Abre tu navegador y visita 
```
http://localhost:3000
```
